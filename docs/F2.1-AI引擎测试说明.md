## 安装命令

```bash
cd orin-ai-engine

# 使用 Poetry 安装
poetry install

# 或使用 venv（已有虚拟环境）
source venv/bin/activate
```

## 运行测试

```bash
# 激活虚拟环境
source venv/bin/activate

# 运行所有测试
python -m pytest tests/ -v

# 运行特定测试文件
python -m pytest tests/test_llm_handler.py -v

# 运行特定测试
python -m pytest tests/test_llm_handler.py::test_llm_handler_success -v
```
