# 前端全局交互规范

> F3.3: 为全局错误提示、空态、加载态统一交互

## 消息提示规范

### ElMessage 用法统一

```javascript
// 成功
ElMessage.success('操作成功')

// 警告
ElMessage.warning('请填写完整信息')

// 错误
ElMessage.error('操作失败，请重试')

// 信息
ElMessage.info('系统维护中')
```

### 错误码映射

| 错误码 | 提示文案 | 类型 |
|--------|----------|------|
| 401 | 登录已过期，请重新登录 | error |
| 403 | 无权限访问 | error |
| 404 | 资源不存在 | warning |
| 500 | 服务异常，请稍后重试 | error |
| NETWORK_ERROR | 网络连接失败，请检查网络 | error |
| TIMEOUT | 请求超时，请重试 | warning |

## 加载态规范

### 页面级加载

```vue
<template>
  <div v-loading="loading" element-loading-text="加载中...">
    <!-- content -->
  </div>
</template>
```

### 组件级加载

```vue
<el-skeleton :loading="loading" animated>
  <template #template>
    <el-skeleton-item variant="h3" style="width: 50%" />
    <el-skeleton-item variant="text" style="margin-top: 10px" />
  </template>
  <!-- actual content -->
</el-skeleton>
```

## 空态规范

### 通用空态

```vue
<el-empty description="暂无数据">
  <el-button type="primary">新建</el-button>
</el-empty>
```

### 按场景分类

| 场景 | 描述文案 | 操作按钮 |
|------|----------|----------|
| 无列表数据 | 暂无记录 | 新建 |
| 无搜索结果 | 未找到相关结果 | 清除筛选 |
| 无权限 | 无权限访问此内容 | - |
| 功能开发中 | 该功能正在开发中 | - |
| 网络异常 | 网络连接失败 | 刷新重试 |

## 统一工具函数

建议创建 `src/utils/message.js` 统一出口：

```javascript
// src/utils/message.js
import { ElMessage } from 'element-plus'

export const message = {
  success: (msg) => ElMessage.success(msg),
  warning: (msg) => ElMessage.warning(msg),
  error: (msg) => ElMessage.error(msg),
  info: (msg) => ElMessage.info(msg)
}

export const handleApiError = (error) => {
  const code = error?.response?.status || error?.code
  const msgMap = {
    401: '登录已过期，请重新登录',
    403: '无权限访问',
    404: '资源不存在',
    500: '服务异常，请稍后重试'
  }
  message.error(msgMap[code] || '操作失败')
}
```
