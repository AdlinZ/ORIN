# 后端接口能力状态标记约定

> 为后端关键接口补统一的能力状态标记或注释约定。

## 状态标记

在 Controller 和 Service 层使用统一注释标记：

```java
// ========== 能力状态标记 ==========
// @Status: VERIFIED - 已验证可交付
// @Status: BETA - 已测试但边界不清晰
// @Status: DEV - 开发中，不保证可用
// @Status: DEPRECATED - 已废弃，不推荐使用
// ================================
```

## 标记位置

- Controller 方法签名上方
- Service 核心方法签名上方
