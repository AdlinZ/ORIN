# 统一网关对外接入快速指南

## 1. 网关地址

- 本地开发：`http://localhost:8080/v1`
- 生产环境：`https://<your-domain>/v1`

## 2. 创建 API Key

1. 登录管理端，进入 API Key 管理页面创建密钥。  
2. 只会显示一次明文密钥（形如 `sk-orin-...`），请妥善保存。

## 3. 健康检查与模型列表

```bash
curl http://localhost:8080/v1/health
curl -H "Authorization: Bearer <YOUR_API_KEY>" \
  http://localhost:8080/v1/models
```

## 4. 聊天接口（OpenAI 兼容）

### curl 示例

```bash
curl -X POST http://localhost:8080/v1/chat/completions \
  -H "Authorization: Bearer <YOUR_API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "Qwen/Qwen2.5-7B-Instruct",
    "messages": [
      {"role":"user","content":"你好，请用一句话介绍你自己。"}
    ],
    "max_tokens": 128,
    "temperature": 0.7
  }'
```

### Python OpenAI SDK 示例

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-orin-xxxxxxxx",
    base_url="http://localhost:8080/v1",
)

resp = client.chat.completions.create(
    model="Qwen/Qwen2.5-7B-Instruct",
    messages=[{"role": "user", "content": "你好"}],
    max_tokens=128,
)
print(resp.choices[0].message.content)
```

## 5. 常见返回码

- `401`：未提供或无效 API Key  
- `429`：触发限流  
- `503`：无可用 Provider  
- `500`：网关内部错误（可用 `X-Trace-Id` 排查）

## 6. Provider 更新策略

- 启动时自动加载配置中的 Provider  
- 修改“模型系统配置”后，Provider 会自动热刷新，无需重启后端
