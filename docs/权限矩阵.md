## 后端权限控制

### 已实现的权限注解

```java
// ApiKeyController.java
@PreAuthorize("hasRole('ADMIN')")
public ResponseEntity<?> deleteApiKey(...) {}
```

```java
public boolean hasRole(Long userId, String roleCode) {

```

## 前端权限控制

### 菜单级控制

```javascript
// Sidebar.vue - visibleMenus
const visibleMenus = computed(() => {
    return menus.filter(menu => {
        if (menu.requiresAdmin && !isAdmin) return false
        return true
    })
})
```

### 按钮级控制

```vue
<!-- 删除按钮仅管理员可见 -->
<el-button v-if="isAdmin" type="danger" @click="handleDelete">
    删除
</el-button>
```

## 权限检查流程

```
用户登录 → 获取角色 → 加载菜单 → 界面渲染
     ↓
请求接口 → 后端校验 @PreAuthorize → 返回结果
```

---

## 特殊权限说明

| 场景 | 处理方式 |
|------|----------|
| 未登录 | 返回 401，前端跳转登录页 |
| 已登录无权限 | 返回 403，前端提示"无权限" |
| Token 过期 | 返回 401，前端跳转登录页 |
| API Key 无效 | 返回 401，前端提示重新配置 |
