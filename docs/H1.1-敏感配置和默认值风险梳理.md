# H1.1: 敏感配置和默认值风险梳理

## 一、敏感配置清单

### 高风险配置（必须配置）

| 配置项 | 说明 | 风险 |
|--------|------|------|
| `DB_PASSWORD` | 数据库密码 | 空密码或弱密码导致未授权访问 |
| `JWT_SECRET` | JWT签名密钥 | 使用默认/弱密钥可被伪造token |
| `SILICONFLOW_API_KEY` | 第三方API密钥 | 泄露可导致额度被盗用 |
| `REDIS_PASSWORD` | Redis密码 | 空密码且暴露公网可被写入恶意数据 |

### 中风险配置（建议配置）

| 配置项 | 说明 | 风险 |
|--------|------|------|
| `RABBITMQ_USERNAME/PASSWORD` | RabbitMQ凭据 | guest/guest 默认账密风险 |
| `MILVUS_TOKEN` | Milvus认证令牌 | 空可能导致未授权访问 |
| `RAGFLOW_DEFAULT_API_KEY` | RAGFlow API密钥 | 可选但若配置应妥善保管 |
| `LANGFUSE_SECRET_KEY` | 可观测性平台密钥 | 可选但若配置应妥善保管 |

---

## 二、默认值风险

### 危险默认值

| 配置项 | 默认值 | 风险描述 |
|--------|--------|----------|
| `REDIS_PASSWORD` | (空) | 无密码Redis暴露公网可被写入恶意数据 |
| `JWT_SECRET` | (用户需手动配置) | 但容易配置为弱密钥如 `secret` |
| `RABBITMQ_USERNAME` | `guest` | 默认账密广泛知晓 |
| `RABBITMQ_PASSWORD` | `guest` | 默认账密广泛知晓 |
| `MILVUS_TOKEN` | (空) | 无认证Milvus可被未授权访问 |
| `CORS_ALLOWED_ORIGINS` | `localhost:5173` | 生产环境使用localhost有风险 |

### 业务逻辑默认值

| 配置项 | 默认值 | 潜在问题 |
|--------|--------|----------|
| `knowledge.chunk.size` | 500 | 可能不适合所有文档类型 |
| `knowledge.chunk.overlap` | 50 | 可能需要根据场景调整 |
| `resilience4j.circuitbreaker...failure-rate-threshold` | 50% | 50%失败率较高 |
| `orin.task.retry.max` | 3 | 可能不够或过多 |
| `rate.limit.requests` | 100 | 可能需要根据实际负载调整 |

## 三、缺失配置时的行为

| 场景 | 预期行为 | 实际风险 |
|------|----------|----------|
| `SILICONFLOW_API_KEY` 为空 | 使用其他provider或失败 | 可能静默降级难以排查 |
| `REDIS_PASSWORD` 为空 | 连接无密码Redis | 公网暴露时风险高 |
| `MILVUS_HOST` 不可达 | 向量检索失败 | 可能导致知识库功能不可用 |
| `RABBITMQ` 不可达 | 任务队列失败 | 工作流执行中断 |
| `JINA_READER_API_KEY` 为空 | Jina Reader不可用 | 网页转Markdown功能降级 |

## 四、安全加固建议

### 必须项（生产环境）

1. **JWT_SECRET**: 使用 `openssl rand -base64 64` 生成强随机密钥
2. **DB_PASSWORD**: 强密码，最小16位，混合大小写+数字+特殊字符
3. **REDIS_PASSWORD**: 必须设置，禁用公网访问
4. **CORS_ALLOWED_ORIGINS**: 生产环境必须配置实际域名，禁止 `*`
5. **RABBITMQ_PASSWORD**: 修改guest默认密码

### 建议项

1. **MILVUS_TOKEN**: 配置认证令牌
2. **API Keys**: 所有第三方API Key使用环境变量，不要硬编码
3. **Rate Limiting**: 根据实际负载调整 `RATE_LIMIT_REQUESTS`

## 六、错误语义检查

建议确保以下场景有统一错误提示：

| 场景 | 建议错误码/提示 |
|------|----------------|
| API Key为空 | `AUTH_001` - "API密钥未配置" |
| API Key无效 | `AUTH_002` - "API密钥无效或已过期" |
| Redis连接失败 | `CACHE_001` - "缓存服务不可用" |
| Milvus连接失败 | `VECTOR_001` - "向量数据库不可用" |
| JWT签名无效 | `AUTH_003` - "认证令牌无效" |
| CORS配置错误 | `SEC_001` - "跨域请求被拒绝" |
