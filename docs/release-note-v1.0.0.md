## ORIN v1.0.0 发布说明

**发布日期**: 2026-03-28  
**版本类型**: 正式版 (GA)

---

### 📢 公告

ORIN 企业级 AI 智能体管理与监控平台正式发布！

这是一个里程碑版本，标志着 ORIN 从功能开发阶段进入稳定交付阶段。

---

### ✨ 新增功能

#### 智能体管理
- 多模型适配 (OpenAI / SiliconFlow / Dify)
- 智能体创建、配置、监控
- API Key 管理与限流

#### 知识库能力
- 文档上传、解析、分块、向量化
- 向量检索 (Milvus)
- 知识图谱构建与查询
- OCR/ASR 多模态支持

#### 工作流与协作
- 可视化工作流编排
- 串行/并行协作执行
- 执行状态追踪

#### 监控运维
- 全链路追踪 (Trace ID)
- 资源监控 (CPU/内存/令牌)
- 告警通知 (邮件/钉钉/企微)
- 系统维护 (备份/升级/日志)

---

### 🐛 问题修复

- 修复了知识库检索超时问题
- 优化了工作流执行性能
- 完善了权限控制边界

---

### ⚠️ 注意事项

1. 首次部署需配置数据库和 Redis
2. AI 模型需自行配置 API Key
3. 向量检索依赖 Milvus (可选)
