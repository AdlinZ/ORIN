# I1.2: 部署前检查清单

## 一、环境要求

### 系统要求
- [ ] Java 17+
- [ ] Node.js 18+
- [ ] Python 3.11+
- [ ] MySQL 8.0.11+
- [ ] Redis 6+
- [ ] Milvus 2.3+ (可选，用于向量检索)

### 端口占用
| 服务 | 端口 | 说明 |
|------|------|------|
| orin-backend | 8080 | Spring Boot API |
| orin-frontend | 5173 | Vue 3 开发服务器 |
| orin-ai-engine | 8000 | FastAPI AI 引擎 |
| MySQL | 3306 | 数据库 |
| Redis | 6379 | 缓存/会话 |
| Milvus | 19530 | 向量数据库 |

---

## 二、必须配置项（.env）

### 数据库
```bash
DB_HOST=localhost
DB_PORT=3306
DB_NAME=orindb
DB_USERNAME=root
DB_PASSWORD=<强密码，最小16位>
```

### Redis
```bash
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=<强密码，禁止空密码>
```

### JWT
```bash
# 生成命令: openssl rand -base64 64
JWT_SECRET=<64位随机密钥>
JWT_EXPIRATION=86400000
```

### 第三方 API
```bash
SILICONFLOW_API_KEY=<有效API Key>
SILICONFLOW_BASE_URL=https://api.siliconflow.cn/v1
```

### CORS（生产必须配置）
```bash
CORS_ALLOWED_ORIGINS=https://your-domain.com,https://app.your-domain.com
```

## 三、安全检查

### 认证配置
- [ ] JWT_SECRET 已修改为强随机密钥（非默认值）
- [ ] DB_PASSWORD 已修改为强密码
- [ ] REDIS_PASSWORD 已设置（非空）
- [ ] RABBITMQ_PASSWORD 已修改（非 guest/guest）
- [ ] CORS_ALLOWED_ORIGINS 已配置实际域名（禁止 *）

### 权限配置
- [ ] 确认 `/api/v1/agents/**`、`/api/v1/users/**` 等管理端点需要有效认证
- [ ] 确认 excludePathPatterns 中的端点有适当的权限校验
- [ ] API Key 创建和删除需要 ADMIN 角色（`@PreAuthorize` 验证）

### 敏感数据
- [ ] `.env` 文件不在代码仓库中
- [ ] 生产环境使用独立的数据库和 Redis 实例
- [ ] API Keys 存储为 hash 而非明文

## 九、Nginx 配置（如使用）

### 反向代理配置
```nginx
server {
    listen 443 ssl;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /var/www/orin-frontend/dist;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API
    location /api/ {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # AI 引擎
    location /v1/ {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
    }
}
```

## 十一、回滚准备

### 回滚命令
```bash
# 停止服务
pkill -f orin-backend
pkill -f orin-ai-engine

# 恢复上一版本
git checkout v1.x.x
mvn package -DskipTests
java -jar target/orin-backend-1.0.0-SNAPSHOT.jar
```

### 数据库回滚
```bash
# 谨慎操作，可能丢失数据
mysql -h localhost -u root -p orindb -e "DROP DATABASE orindb;"
git checkout v1.x.x
# 重新运行 Flyway 迁移
```

## 十二、常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 后端启动失败 | 端口被占用 | `lsof -i :8080` 检查 |
| 前端构建失败 | Node 版本不对 | 确认 Node 18+ |
| API 401 错误 | JWT/API Key 无效 | 检查认证头和配置 |
| Milvus 连接失败 | 服务未启动 | `docker ps | grep milvus` |
| Redis 连接失败 | 密码错误 | 检查 REDIS_PASSWORD |
