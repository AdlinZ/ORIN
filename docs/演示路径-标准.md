# 标准演示路径

> I3.1: 准备 3 条标准演示路径：智能体、知识库、工作流/协作

---

## 演示 1：智能体聊天

### 场景
创建一个智能体，使用它进行多轮对话

### 步骤

1. **创建智能体**
   - 进入「智能体管理」→「创建智能体」
   - 填写名称："客服助手"
   - 选择模型："deepseek-ai/DeepSeek-V2-Chat"
   - 配置提示词："你是一个热情的客服助手"

2. **发起对话**
   - 进入智能体详情页
   - 输入："你好，我想咨询产品信息"
   - 观察流式响应

3. **多轮对话**
   - 继续追问："这个产品有什么优势？"
   - 观察上下文保持

### 预期结果
- 智能体成功创建
- 流式响应正常输出
- 多轮上下文保持

## 演示 2：知识库检索

### 场景
上传文档，构建知识库，进行向量检索

### 步骤

1. **创建知识库**
   - 进入「知识库」→「创建知识库」
   - 填写名称："产品文档库"
   - 选择向量模型

2. **上传文档**
   - 点击「上传文档」
   - 选择 PDF 文件
   - 等待解析完成

3. **执行检索**
   - 进入知识库详情
   - 输入搜索词："产品功能"
   - 查看检索结果

### 预期结果
- 文档成功上传并解析
- 向量成功存储
- 检索返回相关结果

---

## 演示 3：工作流与协作

### 场景
创建一个工作流，执行串行/并行协作任务

### 步骤

1. **创建工作流**
   - 进入「工作流」→「创建工作流」
   - 添加 LLM 节点
   - 添加 Tool 节点

2. **执行工作流**
   - 点击「执行」
   - 输入测试参数
   - 观察执行过程

3. **协作任务**
   - 创建协作任务
   - 添加多个子任务
   - 选择串行/并行执行

### 预期结果
- 工作流成功执行
- 协作任务按预期执行
- 结果正确汇总
