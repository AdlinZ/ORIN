# 验收演示脚本

> I3.4: 输出一版验收演示脚本

---

## 脚本结构

```
./demo.sh [场景] [选项]

场景:
  normal    - 正常流程演示 (默认)
  error     - 异常场景演示
  admin     - 管理功能演示

选项:
  --skip-login  - 跳过登录步骤
  --quick       - 快速模式 (减少等待)
```

---

## 演示脚本

### normal 场景 (正常流程)

```bash
#!/bin/bash
set -e

echo "=== ORIN 验收演示脚本 - 正常流程 ==="

# 1. 登录
echo "[1/6] 登录..."
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}'
echo ""

# 2. 创建智能体
echo "[2/6] 创建智能体..."
curl -X POST http://localhost:8080/api/v1/agent/create \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"演示智能体","model":"deepseek-ai/DeepSeek-V2-Chat"}'
echo ""

# 3. 创建知识库
echo "[3/6] 创建知识库..."
curl -X POST http://localhost:8080/api/v1/knowledge/create \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name":"演示知识库","description":"用于演示"}'
echo ""

# 4. 执行工作流
echo "[4/6] 执行工作流..."
curl -X POST http://localhost:8080/api/v1/workflow/execute/1 \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"input":"测试输入"}'
echo ""

# 5. 创建协作任务
echo "[5/6] 创建协作任务..."
curl -X POST http://localhost:8080/api/v1/collaboration/create \
  -H "Authorization: Bearer $TOKEN" \
  -d '{"name":"演示任务","mode":"serial"}'
echo ""

# 6. 查看监控数据
echo "[6/6] 查看监控数据..."
curl -http://localhost:8080/api/v1/monitor/stats \
  -H "Authorization: Bearer $TOKEN"
echo ""

echo "=== 演示完成 ==="
```

### error 场景 (异常流程)

```bash
#!/bin/bash
set -e

echo "=== ORIN 验收演示脚本 - 异常流程 ==="

# 1. 不带 Token 访问
echo "[1/3] 未授权访问..."
curl http://localhost:8080/api/v1/agent/list
echo ""

# 2. 使用错误凭证登录
echo "[2/3] 错误凭证登录..."
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"wrong"}'
echo ""

# 3. 调用未配置的 Provider
echo "[3/3] Provider 未配置..."
curl -X POST http://localhost:8080/api/v1/agent/chat \
  -H "Content-Type: application/json" \
  -d '{"agentId":"1","message":"hello"}'
echo ""

echo "=== 异常演示完成 ==="
```

## 验证检查点

运行脚本后，检查以下输出：

| 步骤 | 检查点 |
|------|--------|
| 登录 | 返回 token |
| 创建智能体 | 返回智能体 ID |
| 创建知识库 | 返回知识库 ID |
| 执行工作流 | 返回执行结果 |
| 创建协作任务 | 返回任务 ID |
| 查看监控 | 返回统计数据 |
