# ORIN 用户手册

本文档面向 ORIN 平台的普通用户，介绍智能体对话、工作流使用与知识库检索。

---

## 1. 快速入门

### 1.1 登录

1. 访问 <http://localhost:5173>
2. 输入用户名和密码（由管理员分配）
3. 首次登录后建议进入"我的 API Key"设置访问密钥

### 1.2 角色与默认页面

| 角色 | 默认进入 |
|------|----------|
| 普通用户 | 服务门户（`/portal`）— 智能体对话与服务入口 |
| 运维 | 智能体列表（`/dashboard/applications/agents`） |
| 管理员 | 监控总览（`/dashboard/runtime/overview`） |

---

## 2. 智能体对话

### 2.1 进入智能体工作台

顶部菜单 → **智能体管理** → **智能体工作台**

### 2.2 与智能体对话

1. 在工作台下拉框选择智能体
2. 在输入框输入问题，点击发送
3. 对话历史自动保存在左侧边栏

### 2.3 切换智能体

如果接入了多个智能体，点击左上角智能体选择器切换。当前智能体支持的能力（工具/MCP/知识库）会在对话界面顶部显示。

### 2.4 多轮对话与上下文

智能体支持多轮对话，能记住同一会话内的上下文。如果需要开始新话题，点击左侧边栏的"新对话"。

---

## 3. 工作流使用

### 3.1 查看工作流列表

顶部菜单 → **智能体管理** → **工作流中心**

### 3.2 使用预置工作流

1. 在工作流列表找到需要的工作流
2. 点击"执行"进入执行页面
3. 填写必要的输入参数
4. 点击"提交执行"
5. 在执行记录中查看结果

### 3.3 发起协作任务

1. 进入 **协作看板**（顶部菜单 → 多智能体协同）
2. 点击"新建协作包"
3. 描述任务目标，设定优先级
4. 系统自动分解子任务并分配执行角色
5. 可以在看板中查看各子任务的状态

### 3.4 人工干预（协作包）

如果你是协作包的管理者，可以：
- **跳过**：跳过某个子任务
- **手动完成**：手动标记某子任务完成
- **重试**：重新执行失败的子任务
- **取消**：终止整个协作包

## 4. 知识库检索

### 4.1 进入知识检索

顶部菜单 → **知识库管理** → **知识检索**

### 4.2 检索内容

1. 在检索框输入问题或关键词
2. 系统在已接入的知识库中匹配相关内容
3. 返回结果会标注来源知识库

### 4.3 查看知识资产

顶部菜单 → **知识库管理** → **知识资产**

可以查看当前已接入的知识库列表、文档数量、同步状态。

---

## 5. API Key 管理

### 5.1 普通用户自助入口

访问 <http://localhost:5173/portal/api-keys>

### 5.2 创建 API Key

1. 进入 API Key 管理页面
2. 点击"创建密钥"
3. 填写密钥名称，选择有效期
4. **复制密钥**（只显示一次）
5. 将密钥用于 API 调用

### 5.3 使用 API Key

```bash
# 示例：查询可用的 Agent
curl http://localhost:8080/api/v1/agents \
  -H "Authorization: Bearer sk-orin-your-key-here"
```

### 5.4 管理密钥

- **禁用密钥**：临时关闭某个密钥的访问权限
- **启用密钥**：重新开放已禁用的密钥
- **删除密钥**：永久删除（无法恢复）

## 6. 常见问题

### Q: 无法登录

- 检查用户名和密码是否正确
- 联系管理员确认账户状态
- 清除浏览器缓存后重试

### Q: 智能体回复很慢

- 可能 AI Engine 在处理复杂任务，请稍等
- 如果长时间无响应，检查 AI Engine 是否在线

### Q: 工作流执行失败

- 检查输入参数是否完整
- 查看错误信息中的 `traceId`
- 联系管理员或查看[手册-管理员.md](./手册-管理员.md)中的告警处理

### Q: 知识库检索无结果

- 确认知识库已同步完成（在知识资产页面查看状态）
- 尝试更换关键词或使用更简洁的表述
- 确认该知识库已接入你的智能体
